# Local sound effects

Place your local sound-effect files here. SmartShorts Studio will use them automatically when present.

Expected filenames:

```text
boom.mp3
swoosh.mp3
whoosh.mp3  # optional legacy alias
click.mp3
clap.mp3
laugh.mp3
```

If a file is missing, the renderer creates a synthetic fallback sound inside the FFmpeg filter graph, so clips still render.
Use only royalty-free, licensed, or self-created effects.
